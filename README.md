## Задача 1

### Текст задачи

Получить последовательность из первых цифр чисел, содержащихся в исходном списке

### Алгоритм решения

1) При помощи метода readPositiveInt запрашиваем ввод целого, положительного числа, получаем и проверяем, если введенная строка соответствует критериям, возвращаем в n, иначе повторяем запрос
2) С помощью метода readSelectedMethod запрашиваем выбор метода заполнения массива, проверяем, если это "0" или "1" - возвращаем значение в selectedMethod
3) Передаем в createSeq n и selected method, и исходя из выбранного кол-ва значений и выбранного метода заполнения инициализируем Z последовательность
4) Выводим полученную последовательность
5) При помощи Seq.map применяем функцию для получения первой цифры числа (takeFirst), результат записываем в resultSeq
6) Выводим новую, полученную последовательность

## Задача 2

### Текст задачи

Сколько раз в списке встречается заданное число?

### Алгоритм решения

1) При помощи метода readPositiveInt запрашиваем ввод целого, положительного числа, получаем и проверяем, если введенная строка соответствует критериям, возвращаем в n, иначе повторяем запрос
2) С помощью метода readSelectedMethod запрашиваем выбор метода заполнения массива, проверяем, если это "0" или "1" - возвращаем значение в selectedMethod
3) Передаем в createSeq n и selected method, и исходя из выбранного кол-ва значений и выбранного метода заполнения инициализируем последовательность
4) Выводим полученную последовательность
5) При помощи метода readFloat запрашиваем ввод вещественного числа, получаем и проверяем, если введенная строка соответствует критериям, возвращаем в compnum, иначе повторяем запрос
6) Передаем в countOfMatches Seq и compnum для нахождения в Seq количества чисел compnum. В countOfMatches используется Seq.fold для подсчета количества чисел соответствующих target (compnum). Полученный результат сохраняем в matches
7) Выводим количество чисел в последовательности, соответствующих заданному

## Задача 3
### Текст задачи
Вывести пути ко всем текстовым файлам (*.txt) в указанном каталоге и его подкаталогах
### Алгоритм решения
1)  Запрашиваем у пользователя путь к каталогу
2)  Проверяем существование данного каталога с помощью Directory.Exists
3)  С помощью метода allTextFiles рекурсивно проходим по каталогу, указанному ранее, и каждому его подкаталогу. Если встречаем файлы формата txt - выводим путь к ним
